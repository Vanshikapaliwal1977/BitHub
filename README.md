# BitHub

A modern, dark-themed portfolio and community platform where developers can showcase their projects, write blogs, connect with peers, and explore opportunities. Built with React, TypeScript, and Tailwind CSS.

- **User Authentication**: Secure login and registration with a sleek dark theme
- **Project Showcase**: Add and display your coding projects with rich details
- **Blog Platform**: Write and share technical blogs with the community
- **Opportunities Board**: Post and discover job opportunities, freelance gigs, and collaborations
- **Community Chat**: Engage in discussions with fellow developers
- **Feedback System**: Collect and provide feedback on projects
- **Responsive Design**: Optimized for desktop and mobile devices
- **Dark Theme**: Modern UI with consistent dark aesthetics

## üì¶ Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/bithub.git
   cd bithub
   ```

2. **Install dependencies**
   ```bash
   npm install
   cd server && npm install
   ```

3. **Set up environment variables**
   - Create `.env` file in the root directory
   - Add necessary environment variables (e.g., database URL, email credentials)

4. **Start the development server**
   ```bash
   # Start the frontend
   npm run dev

   # In another terminal, start the backend
   npm run server
   ```

5. **Open your browser**
   - Navigate to `http://localhost:5173` for the frontend
   - Backend runs on `http://localhost:5000`
